# Optimizer Module

优选模块 - 为前端提供多源数据聚合与优选服务

## 目录结构

```
optimizer/
├── core/           # 核心基类、工厂、服务管理器
├── services/       # 具体的优选服务实现
├── routes/         # API路由
├── config/         # 配置文件
├── utils/          # 工具函数
└── README.md       # 本文件
```

## 功能说明

本模块负责从多个后端数据源拉取复杂信息，进行聚合、筛选、优选后返回给前端。
