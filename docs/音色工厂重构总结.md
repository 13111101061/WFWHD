## 配置驱动实现

### 新的API调用方式

#### 方式1：使用系统ID（推荐）
```javascript
POST /api/tts/synthesize
{
  "text": "你好世界",
  "systemId": "cosyvoice-longxiaochun"
}
```

后端自动转换为：
```javascript
{
  "provider": "aliyun",
  "serviceType": "cosyvoice",
  "voice": "longxiaochun_v2",
  "model": "cosyvoice-v2"
}
```

#### 方式2：直接传服务商参数（兼容旧版）
```javascript
POST /api/tts/synthesize
{
  "text": "你好世界",
  "service": "aliyun_cosyvoice",
  "voice": "longxiaochun_v2"
}
```

## 性能影响

**删除缓存的影响：**
- 每次请求都需要调用服务商API
- 响应时间会增加（取决于服务商）
- 存储空间占用减少（不保存缓存文件）

**收益：**
- 代码逻辑更清晰
- 维护成本降低
- 避免缓存一致性问题
